# La Hamburguesa Loca Web Logo Assets

Includes:
- Transparent cutout PNG logo sizes: 16, 32, 48, 64, 96, 128, 152, 180, 192, 256, 384, 512, 1024
- WebP versions for larger web usage
- favicon.ico with 16x16, 32x32, and 48x48 embedded
- site.webmanifest
- html-head-snippet.html

Recommended usage:
- Navbar/logo: `la-hamburguesa-loca-logo-256x256.webp` or `512x512.webp`
- Social/profile image: `la-hamburguesa-loca-logo-1024x1024.png`
- Apple touch icon: `la-hamburguesa-loca-logo-180x180.png`
- Favicon: `favicon.ico`
